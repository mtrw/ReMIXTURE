# ReMIXTURE
The ReMIXTURE method implementation in R, for Tripodi &amp; Rabanus-Wallace et al.

A reference to the publication describing the method will be added upon acceptance of the manuscript.

The script implements the method and produces plots as appear in the release publication. It can be run interactively. Instructions are given as comments, regarding where parameters can be adjusted and which data strutures are required to run the procedure.

For support, or if you are interested in expanding, improving, re-writing, or properly producing a package from this implementation, please email Tim at mtrw85@gmail.com.
